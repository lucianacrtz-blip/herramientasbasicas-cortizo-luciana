# Trabajo-Final-Herramientas-basicas-analisis-de-Datos
El presente Trabajo Final se basa en el análisis de los precios de medicamentos de PAMI (Publicación vigente al 1/04/2026) y explora cuales son los principios activos,  el % de cobertura y laboratorios elaboradores de los medicamentos de mayor y menor costo para el paciente.
